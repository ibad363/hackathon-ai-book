# Research for RAG Chatbot

## Research Tasks

- **FastAPI Best Practices**:
  - Research best practices for structuring a FastAPI application.
  - Investigate optimal patterns for dependency injection and background tasks.
  - Explore recommended methods for security, including API key management and CORS configuration.

- **Docusaurus Integration**:
  - Research how to create and inject a custom React component (floating chat button) into all pages of a Docusaurus site.
  - Investigate "swizzling" in Docusaurus to override theme components.
  - Explore state management for the chat widget within the Docusaurus environment.

- **RAG Pipeline Best Practices**:
  - Research optimal chunking strategies for Markdown files, focusing on preserving context around headings.
  - Investigate best practices for using the Gemini API for embeddings and chat generation.
  - Explore retrieval strategies beyond simple top-k similarity search (e.g., MMR).

- **Qdrant Cloud**:
  - Research best practices for creating and managing collections in Qdrant Cloud.
  - Investigate filtering and payload indexing for efficient metadata-based searches.

- **Neon Serverless Postgres**:
  - Research best practices for schema design for storing chat history.
  - Investigate connection pooling and management in a serverless environment.
- **OpenAI Agents SDK**:
  - Research best practices for using the OpenAI Agents SDK with the Gemini API.
  - Investigate how to enforce strict "answer only from context" rules.
  - Explore patterns for handling conversational history and follow-up questions.
